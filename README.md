# Inventory_management_System_Nirnay
CLI-based Inventory Management System built with Python — demonstrates OOP, layered architecture, and SOLID principles.
A command-line Inventory Management System built in Python as part of a structured OOP learning path. The project models real-world inventory operations — adding, updating, deleting, and searching products — while following clean architecture principles: models handle data, services handle business logic, and utils handle persistence. Data is stored locally in JSON. Built to practice Classes, Encapsulation, Abstraction, Magic Methods, and the Single Responsibility Principle in a practical, hands-on way.
